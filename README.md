## MR Movie (Reactjs Project)

▶️ <a href="https://yourmrmovies.netlify.app/">MR Movie</a> is created using:
 - React js
 - API Integration (Movie API) 
 - Hook (useState, useEffect)


 


## Installation

Install all dependencies

```bash
  npm install
```
Run 

```bash
  npm start
```
